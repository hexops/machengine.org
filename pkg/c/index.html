C libraries & headers packaged via the Zig build system

Mach provides an ecosystem of C libraries via the Zig package manager.

These arenâ€™t Zig bindings to these libraries (which we have separately), but instead are just forks of the actual project with their build system replaced by build.zig so you can depend on them and build them using the Zig package manager.

Pure forks with build.zig:

  • brotli
  • harfbuzz
  • freetype
  • glfw
  • basisu
  • opus
  • opusfile
  • opusenc
  • flac
  • ogg
  • spirv-tools
  • spirv-cross

Header packages:

  • direct3d-headers
  • wayland-headers
  • x11-headers
  • vulkan-headers
  • opengl-headers
  • linux-audio-headers

Special cases:

  • vulkan-zig-generated pre-generated Vulkan bindings
  • mach-objc Objective C API bindings generated by mach-objc-generator
  • xcode-frameworks macOS XCode frameworks
  • dawn (wrench-generated source branches with vendored dependency tree)
  • DirectXShaderCompilerwn (fork for Zig compilation support)
